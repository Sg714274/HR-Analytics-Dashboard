# HR-Analytics-Dashboard
This project presents an in-depth HR Analytics Dashboard designed to analyze employee attrition, identify major contributing factors, and generate insights that help HR teams make better workforce decisions.

Project Overview

The objective of this project is to analyze employee attrition data and reveal patterns that HR teams can use to:

Reduce attrition

Improve employee engagement

Enhance recruitment strategies

Strengthen workplace satisfaction

This dashboard visualizes key HR metrics such as attrition count, attrition rate, age distribution, satisfaction scores, job role analysis, and salary-level insights.
